## Design-Prinzipien

### Farben & Kontraste
- ✅ WCAG 2.1 AA Standard für Text-Kontraste
- ✅ Primärfarben für wichtige CTAs
- ✅ Akzentfarbe (Orange) für Newsletter

### Spacing
- Mobile: 1rem Padding
- Tablet: 1.5rem Padding  
- Desktop: 2rem Padding
- Gap: 1-2rem je nach Kontext

### Typografie
- Minimum: 16px Base-Size
- Maximale Zeilen-Länge: 75 Zeichen
- Line-Height: 1.6 für Fließtext

### Interaktive Elemente
- Minimum Touch-Target: 44x44px
- Hover-States auf Desktop
- Focus-States für Accessibility
- Smooth Transitions (0.3s)

---

## Interaktionen

### Navigation
- Mobile: Toggle-Menu (nicht implementiert, vorerst vertikal)
- Tablet/Desktop: Horizontale Navigation
- Active State für aktuelle Seite

### Filter (Artikel-Seite)
1. Klick auf Filter-Button
2. Artikel werden gefiltert
3. Smooth Transition
4. Active State auf Button

### Formulare
1. Focus States bei Eingabe
2. Validation (HTML5)
3. Submit Animation
4. Success/Error Messages
